# HRMS_Backend
# Employee Management & Attendance API

## Project Overview

This project is a **FastAPI-based backend application** that provides APIs for managing employees and their attendance.
The system allows an admin to create employees, view employee records, delete employees, and manage attendance.

The application follows a **clean architecture pattern (Router → Service → Repository → Database)** to ensure maintainability, scalability, and separation of concerns.

Features included in this project:

* Create a new employee
* View all employees
* Delete an employee
* Mark employee attendance
* View attendance records
* Pagination support for employee listing
* Structured API responses (status, message, data)
* MySQL database integration
